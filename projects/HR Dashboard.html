üîß Analytical Approach

1. Data Cleaning (Excel / Power Query)

- Removed null values in Age, Salary, Join Date
- Standardized Job Title categories
- Created Age Bins (20-30, 30-40, etc.)
- Converted text dates to proper date format

2. DAX Measures for KPIs

Headcount = COUNT(Employee[EmployeeID])

Avg Salary = AVERAGE(Employee[Salary])

Avg Leave Balance = AVERAGE(Employee[LeaveBalance])

LBL over 20 = CALCULATE(COUNT(Employee[EmployeeID]), Employee[LeaveBalance] > 20)

3. Trend Analysis

Cumulative Headcount = CALCULATE( COUNT(Employee[EmployeeID]), FILTER(ALL(Employee), Employee[JoinDate] <= MAX(Employee[JoinDate])) )
